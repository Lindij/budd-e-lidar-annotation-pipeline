# Dataset Card: BUDD-e LIDAR

## Summary
Hospital mobile robot LIDAR ROS bag recordings for 3D point cloud labeling and tracking.

## Data Sources
- Sensor: LIDAR on mobile robot
- Format: ROS bag .bag
- Environment: hospital

## Structure
- Raw bags: data/raw/rosbags
- Derived point clouds: data/interim, data/processed
- Labels: data/labels

## Privacy and Ethics
- Ensure no patient-identifiable information is present in labels or exports.
